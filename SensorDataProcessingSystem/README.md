# Sensor Data Processing & Fault Analysis System (Java)

## Description
A Java-based console application that processes sensor data such as
voltage readings. It applies searching and sorting algorithms to analyze
data and detect abnormal values, inspired by battery monitoring systems.

## Concepts Used
- Java
- Arrays & Strings
- Linear & Binary Search
- Sorting Algorithms
- Time Complexity
- OOP Concepts

## Real-Life Use Case
- Battery Management Systems (BMS)
- IoT Sensor Data Analysis
- Fault Detection Systems
